# HikRobot-HikCamera-Tool

#### 通过封装常用相机操作，让你的工业相机开箱即用！
本示例适用于 **海康机器人MV系列** 工业相机产品
> 最新版本已通过测试的型号:
>
>MV-CA013-21UC
> 
> MV-CA016-10UC
